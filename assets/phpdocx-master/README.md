# phpdocx
PHP Docx generator free

# Using (composer.json)
```
"require": {
        "lpkapil/phpdocx": "1.0.0"
    },
    "repositories": [
        {
            "type": "package",
            "package": {
                "name": "lpkapil/phpdocx",
                "version": "1.0.0",
                "source": {
                    "url": "https://github.com/lpkapil/phpdocx.git",
                    "type": "git",
                    "reference": "master"
                }
            }
        }
    ]
```
